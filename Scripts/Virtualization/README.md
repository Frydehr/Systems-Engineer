### 🖥️ Virtualization & Clustering
* **`New-ClusteredVM.ps1`**: Automates the deployment of highly available VMs on Hyper-V clusters.
* **`Get-VMResourceReport.ps1`**: Audits CPU, RAM, and Disk over-provisioning across all running guests.
* **`Invoke-VMLiveMigration.ps1`**: Orchestrates the bulk movement of VMs between cluster nodes for maintenance.
* **`Test-VMReplicationHealth.ps1`**: Monitors Hyper-V Replica status to ensure DR synchronicity.
* **`Optimize-VHDXSize.ps1`**: Mounts and compacts dynamic virtual hard disks to reclaim host space.
* **`Get-OrphanedVirtualDisks.ps1`**: Scans CSVs/Datastores for VHDX files not attached to any VM.
