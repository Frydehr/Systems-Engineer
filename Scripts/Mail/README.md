📧 Mail & Notifications Suite
* **`Send-BasicEmailNotification.ps1`**: Quick SMTP-based alerts and notifications.
* **`Send-HTMLReport.ps1`**: Converts PowerShell objects into clean HTML tables for reporting.
* **`Send-CriticalAlert.ps1`**: High-priority alerting for system failures and on-call notification.
* **`Get-MailQueueStatus.ps1`**: Monitors local SMTP relay or Exchange queues for stuck messages.
* **`Send-AttachmentReport.ps1`**: Automates the distribution of log files or CSVs as email attachments.
